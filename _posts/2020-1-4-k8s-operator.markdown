    开始深入了解 Kubernetes Operators。

## 概念

Operator 是一种打包、部署和管理 Kubernetes 应用程序的一种方式。Kubernetes 应用程序是既部署在 Kubernetes 上，又使用 Kubernetes API 和 kubectl 工具进行管理的应用程序。
